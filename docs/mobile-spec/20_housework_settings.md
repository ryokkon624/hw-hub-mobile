# #20 家事設定一覧 モバイル仕様

## 1. 概要

この世帯の家事マスタ（テンプレート）を一覧表示し、新規作成・編集へ遷移する画面。Web版SP版と同等の機能をそのまま踏襲する。

---

## 3. 機能仕様

### 3.1 一覧表示

- 現在選択中の世帯の家事マスタを取得して表示
- 世帯切り替え時は一覧を再取得
- 各カードに「家事名」「カテゴリバッジ」「周期サマリー」「デフォルト担当者」「有効期間」を表示
- タップで #22 家事編集 へ遷移

### 3.2 カテゴリフィルタ

| 選択肢 | 内容 |
|---|---|
| すべて | フィルタなし（デフォルト） |
| 掃除 | `CLEAN` |
| キッチン | `KITCHEN` |
| ガーデン | `GARDEN` |
| ゴミ | `GARBAGE` |
| ペット | `PET` |

### 3.3 カテゴリバッジ色

| カテゴリ | 色 |
|---|---|
| 掃除（CLEAN） | 青 |
| キッチン（KITCHEN） | オレンジ |
| ガーデン（GARDEN） | シアン |
| ゴミ（GARBAGE） | グリーン |
| ペット（PET） | パープル |

### 3.4 周期サマリー表示

| 周期タイプ | 表示例 |
|---|---|
| WEEKLY | 毎週 月・水・金 |
| MONTHLY | 毎月 15 日 / 毎月末日 |
| NTH_WEEKDAY | 第2 火曜日 |

### 3.5 ページネーション

- 10件/ページ
- フィルタ適用後の件数に基づいてページを計算

### 3.6 新規作成

- 「家事を追加する」タップで #21 家事新規作成 へ遷移

> 💡 家事マスタに削除機能はない。使用しなくなった家事は有効終了日を現在日付より前に設定する運用。

---

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/houseworks?householdId={householdId}` | 家事マスタ一覧取得 |
| GET | `/api/households/{householdId}/members` | メンバー一覧（デフォルト担当者名解決用） |

---

## 5. 画面遷移

```
[家事設定一覧]
  ├─ ← 戻る ────────────→ 設定トップ (#17)
  ├─ 家事を追加する ─────→ #21 家事新規作成
  └─ 各家事カードタップ ─→ #22 家事編集
```
