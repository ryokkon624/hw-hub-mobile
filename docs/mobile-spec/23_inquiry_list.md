# #23 問い合わせ一覧 モバイル仕様

## 1. 概要

ユーザーが送信したサポート問い合わせの一覧表示画面。Web版SP版と同等の機能をそのまま踏襲する。

---

## 3. 機能仕様

### 3.1 一覧表示

- `GET /api/inquiries` でログインユーザーの問い合わせ一覧を取得
- 各アイテムに「件名（#ID: タイトル）」「カテゴリバッジ」「ステータスバッジ」「作成日時」を表示
- タップで #25 問い合わせ詳細 へ遷移
- 空の場合は「まだお問い合わせがありません」を表示

### 3.2 カテゴリバッジ色

| カテゴリ | 色 |
|---|---|
| 一般（GENERAL） | グレー |
| 家事（HOUSEWORK） | アンバー |
| 買い物（SHOPPING） | グリーン |
| アカウント（ACCOUNT） | ブルー |
| バグ（BUG） | レッド |

### 3.3 ステータスバッジ色

| ステータス | 色 | 表示ラベル |
|---|---|---|
| OPEN | ブルー | 受付中 |
| AI_ANSWERED | バイオレット | AI回答済み |
| PENDING_STAFF | アンバー | スタッフ対応中 |
| STAFF_ANSWERED | グリーン | スタッフ回答済み |
| CLOSED | グレー | 解決済み |

### 3.4 新規作成

- 「新規お問い合わせ」タップで #24 問い合わせ新規作成 へ遷移

---

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/inquiries` | 問い合わせ一覧取得 |

