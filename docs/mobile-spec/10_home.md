# #10 ホーム モバイル仕様

## 1. 概要

アプリのダッシュボード画面。世帯への所属状況に応じて表示内容が変わる。

- **世帯未所属**: オンボーディングカード + 各機能カード（件数はゼロ）
- **世帯所属済み**: 各機能のサマリーカード4枚

## 3. 機能仕様

### 3.1 My Tasksカード

| 要素 | 内容 |
|---|---|
| 今日の家事件数 | 今日が実施日の自分担当タスク件数 |
| 直近1週間 | 今日から7日以内の自分担当タスク件数 |
| 期限超過 | 今日より前の未対応タスク件数（赤字表示） |
| My Tasks を開くボタン | #12 My Tasks へ遷移 |

### 3.2 家事の未割り当てカード

| 要素 | 内容 |
|---|---|
| 未割り当て（全期間） | 担当者未設定のタスク総件数 |
| 今日〜3日以内 | 今日から3日以内に実施予定の未割り当て件数 |
| 家事割り当てを開くボタン | #11 家事分担へ遷移 |

### 3.3 買い物リストカード

| 要素 | 内容 |
|---|---|
| 購入場所別件数 | 未購入アイテムを購入場所ごとに件数表示 |
| 直近2日以内に追加されたアイテム | 件数表示 |
| 買い物リストを開くボタン | #13 買い物リストへ遷移 |

### 3.4 おうちの様子カード（グラフ）

| 要素 | 内容 |
|---|---|
| グラフ種別 | 積み上げ棒グラフ（fl_chart: `BarChart`） |
| 表示期間 | 過去6日〜未来6日（計13日分） |
| X軸 | 日付（M/d形式） |
| Y軸 | タスク件数 |
| 色分け | メンバーごと + 未割当 |
| 凡例 | グラフ下部にメンバー名と色を表示 |

**レイアウト方針:**
- まずは画面幅に収める形で実装
- 実機確認でラベルが重なる場合は横スクロール対応に切り替える

### 3.5 オンボーディングカード（世帯未所属時のみ）

| ボタン | 遷移先 |
|---|---|
| おうち設定へ移動 | #19 世帯設定 |
| 家事設定へ移動 | #20 家事設定一覧 |

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/households/{householdId}/members` | メンバー一覧（おうちの様子グラフ凡例用） |
| GET | `/api/houseworks?householdId={householdId}` | 家事マスタ一覧（タスク名解決用） |
| GET | `/api/housework-tasks?householdId={householdId}&status=0` | 未対応タスク（My Tasks / 家事未割り当てカード / おうちの様子） |
| GET | `/api/housework-tasks?householdId={householdId}&status=1` | 完了タスク（おうちの様子グラフ） |
| GET | `/api/households/{householdId}/shopping-items` | 買い物リストカード |

## 5. 状態管理

```
HomeState
├── myTasksSummary: MyTasksSummary        // My Tasksカード用
│   ├── todayCount: int
│   ├── weekCount: int
│   └── overdueCount: int
├── unassignedSummary: UnassignedSummary  // 家事未割り当てカード用
│   ├── totalCount: int
│   └── urgentCount: int                  // 今日〜3日以内
├── shoppingItems: List<ShoppingItem>     // 買い物リストカード用
├── householdOverview: List<DailyOverview>// おうちの様子グラフ用
├── hasHousehold: bool                    // 世帯所属状況
└── isLoading: bool
```
