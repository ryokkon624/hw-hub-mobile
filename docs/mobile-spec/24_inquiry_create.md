# #24 問い合わせ新規作成 モバイル仕様

## 1. 概要

サポート問い合わせを新規作成する画面。Web版SP版と同等の機能をそのまま踏襲する。

---

## 3. 機能仕様

### 3.1 フォームフィールドと検証ルール

| フィールド | 必須 | ルール |
|---|---|---|
| カテゴリ | ✓ | GENERAL / HOUSEWORK / SHOPPING / ACCOUNT / BUG |
| 件名 | ✓ | 最大200文字 |
| お問い合わせ内容 | ✓ | 任意の文字数 |

### 3.2 カテゴリ選択肢

| 値 | 表示ラベル |
|---|---|
| GENERAL | 一般 |
| HOUSEWORK | 家事 |
| SHOPPING | 買い物 |
| ACCOUNT | アカウント |
| BUG | バグ報告 |

### 3.3 送信

- バリデーション通過後に `POST /api/inquiries`
- レスポンス `{ inquiryId }` を受け取り、成功後スナックバーで通知
- 作成した問い合わせの詳細画面（#25 問い合わせ詳細）へ遷移

### 3.4 キャンセル

- #23 問い合わせ一覧 へ戻る

---

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| POST | `/api/inquiries` | 問い合わせ作成 |

**リクエストボディ:**
```json
{
  "category": "GENERAL",
  "title": "件名テキスト",
  "body": "お問い合わせ内容テキスト"
}
```

**レスポンス:**
```json
{
  "inquiryId": 123
}
```

---

## 5. 画面遷移

```
[問い合わせ新規作成]
  ├─ キャンセル ─────────→ #23 問い合わせ一覧
  └─ 送信成功 ───────────→ #25 問い合わせ詳細（作成したID）
```
