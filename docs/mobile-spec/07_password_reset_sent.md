# #7 リセットメール送信 モバイル仕様

## 1. 概要

パスワードリセットメールの送信完了を伝える案内画面。

送信先メールアドレスを表示し、メールが届かない場合の再送ボタンと別メールアドレスでのやり直し導線を提供する。

Web版との差異はなし。

## 3. 機能仕様

### 3.1 メールアドレスの引き渡し

- パスワード忘れ画面 (#6) からナビゲーション引数でメールアドレスを受け取る
- メールアドレスがない場合は「もう一度送信する」タップ時にパスワード忘れ画面 (#6) へリダイレクト

### 3.2 再送処理

- 「もう一度送信する」タップで `POST /api/auth/password-reset/request` を再呼び出し
- 送信中はスピナー表示（非活性）
- 再送成功時: スナックバーで「メールを再送しました」と通知
- 再送失敗時: フォーム下部にエラーメッセージを表示

### 3.3 別メールアドレスでやり直し

- 「別のメールアドレスを使う」タップ → パスワード忘れ画面 (#6) へ遷移

---

## 4. APIエンドポイント

| メソッド | エンドポイント | 用途 |
|---|---|---|
| POST | `/api/auth/password-reset/request` | パスワードリセットメール再送 |

### POST /api/auth/password-reset/request

```json
// Request
{ "email": "user@example.com" }

// Response 204 No Content
```

## 5. 画面遷移

```
[リセットメール送信完了画面]
  │
  ├─ もう一度送信する ────────────→ （同画面でAPI再呼び出し）
  ├─ 別のメールアドレスを使う ────→ パスワード忘れ (#6)
  └─ ログイン画面へ戻る ──────────→ ログイン (#1)
```
