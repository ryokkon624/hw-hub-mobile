# #21 家事新規作成 モバイル仕様

## 1. 概要

家事マスタを新規作成する画面。Web版SP版と同等の機能をそのまま踏襲する。テンプレートから入力値を一括反映できる「テンプレート選択」機能を持つ。

## 3. 機能仕様

### 3.1 テンプレート選択（モーダル）

- 「テンプレートから選択」タップで `GET /api/housework-templates` を呼び出し、テンプレート一覧をモーダル表示
- カテゴリフィルタでテンプレートを絞り込み可能
- テンプレートを選択すると以下フィールドを一括上書き:
  - 家事名、説明、カテゴリ、周期タイプ、曜日/日付/第n週/曜日
- 選択後、推薦メモ（`recommendation`）があればバナーで表示（×で閉じる）

### 3.2 フォームフィールドと検証ルール

| フィールド | 必須 | ルール |
|---|---|---|
| 家事名 | ✓ | 最大100文字 |
| 説明 | — | 任意テキスト |
| カテゴリ | ✓ | CLEAN / KITCHEN / GARDEN / GARBAGE / PET |
| 周期タイプ | ✓ | WEEKLY / MONTHLY / NTH_WEEKDAY |
| 曜日（WEEKLY時） | ✓ | 1曜日以上選択 |
| 月の日付（MONTHLY時） | ✓ | 1〜31（31=月末） |
| 第n週（NTH_WEEKDAY時） | ✓ | 1〜5 |
| 曜日（NTH_WEEKDAY時） | ✓ | 0〜6 |
| デフォルト担当者 | — | 世帯メンバーから選択（未設定可） |
| 開始日 | ✓ | YYYY-MM-DD |
| 終了日 | ✓ | YYYY-MM-DD、開始日以降 |

### 3.3 周期タイプによる動的表示

- WEEKLY: 曜日チェックボックスを表示
- MONTHLY: 月の日付セレクト（1〜30、31=月末）を表示
- NTH_WEEKDAY: 第n週セレクト + 曜日セレクトを表示

### 3.4 保存

- バリデーション通過後に `POST /api/houseworks`
- 成功後スナックバーで通知し、#20 家事設定一覧 へ戻る

---

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/housework-templates` | テンプレート一覧取得 |
| GET | `/api/households/{householdId}/members` | デフォルト担当者選択肢用 |
| POST | `/api/houseworks` | 家事マスタ作成 |
