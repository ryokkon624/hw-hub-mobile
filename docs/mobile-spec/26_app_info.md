# #26 アプリ情報 モバイル仕様

## 1. 概要

アプリのバージョン情報・利用規約・プライバシーポリシー・開発者情報・OSSライセンス概要を表示する画面。Web版SP版と同等の機能をそのまま踏襲する。

## 3. 機能仕様

### 3.1 バージョン情報

| 項目 | 取得方法 |
|---|---|
| フロントエンド（アプリ）バージョン | ビルド時に `pubspec.yaml` の `version` を静的埋め込み |
| APIバージョン | 起動時に `GET /actuator/info` を呼び出し `app.version` を表示。取得中は「取得中...」、失敗時は「不明」を表示 |

### 3.2 利用規約・プライバシーポリシー

- 「利用規約」タップで #27 利用規約 へ遷移
- 「プライバシーポリシー」タップで #28 プライバシーポリシー へ遷移

### 3.3 データとプライバシー概要（静的表示）

以下3点を箇条書きで表示:
- 世帯データは所属メンバーのみに公開される
- 改善目的の匿名データを収集する
- データを第三者へ販売しない

### 3.4 開発者情報（静的表示）

- アプリ名
- 運営者名
- 問い合わせ先メールアドレス

### 3.5 OSSライセンス（静的表示）

- 利用しているOSSライセンスの概要テキストを表示
