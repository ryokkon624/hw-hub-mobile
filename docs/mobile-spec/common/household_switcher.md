# 共通仕様: 世帯切り替え（HouseholdSwitcher）

## 1. 概要

複数の世帯に所属しているユーザーが、操作対象の世帯を切り替えるための共通UI。
1世帯のみ所属の場合は表示しない。

---

## 2. 表示条件

| 所属世帯数 | 表示 |
|---|---|
| 1世帯のみ | 非表示（ボトムナビのみ） |
| 2世帯以上 | 世帯インジケーターバーを表示 |

---

## 3. UI構成

### 3.1 世帯インジケーターバー

ボトムナビゲーションの直上に固定表示する。

```
┌────────────────────────────────────┐
│                                    │
│           コンテンツ                │
│                                    │
├────────────────────────────────────┤
│  自宅 ▼                            │  ← 世帯インジケーターバー（複数世帯時のみ）
├────────────────────────────────────┤
│ 🏠    📋    ✅    🛒    ⚙️         │
│ホーム 割当  Tasks 買物  設定        │
└────────────────────────────────────┘
```

**表示内容**
- 現在選択中の世帯名
- ▼アイコン（タップで切り替え可能であることを示す）

### 3.2 世帯切り替えボトムシート

世帯インジケーターバーをタップすると、ボトムシートが開く。

```
┌────────────────────────────────────┐
│                                    │
│  世帯を選択                         │
│                                    │
│  ✅  自宅                           │  ← 現在選択中（チェック表示）
│      自宅                           │
│                                    │
│      実家                           │  ← 切り替え先
│      実家                           │
│                                    │
│ ────────────────────────────────── │
│           キャンセル                 │
└────────────────────────────────────┘
```

**表示内容**
- 所属する全世帯をリスト表示
- 現在選択中の世帯にチェックマーク
- 各行: 世帯名 + 世帯の説明

---

## 4. 動作仕様

### 4.1 世帯切り替え時の挙動

1. ボトムシートで別世帯をタップ
2. ボトムシートを閉じる
3. 現在表示中の画面のデータを選択した世帯のデータで再取得・再描画
4. 世帯インジケーターバーの世帯名を更新

### 4.2 選択中世帯の永続化

- 選択した世帯はアプリ再起動後も維持する
- 保存先: `SharedPreferences`（`selected_household_id`）
- ただしその世帯への所属が解除された場合はデフォルト世帯にフォールバック

---

## 5. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/households/me` | 所属世帯一覧取得 |

> ⚠️ エンドポイントはWeb版の実装を確認の上、正確なパスに修正すること

---

## 6. 状態管理

アプリ全体で共有するグローバルな状態として管理する。

```
HouseholdState
├── households: List<Household>   // 所属世帯一覧
├── selectedHousehold: Household  // 現在選択中の世帯
└── isMultiple: bool              // 複数世帯所属かどうか（インジケーター表示制御）
```

本仕様を適用する画面は以下の通り。複数世帯所属ユーザーの場合、これら全画面で世帯インジケーターバーが表示される。
