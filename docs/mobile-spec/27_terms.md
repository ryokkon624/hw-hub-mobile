# #27 利用規約 モバイル仕様

## 1. 概要

アプリの利用規約を表示する静的コンテンツ画面。Web版SP版と同等の内容をそのまま踏襲する。APIコールなし。

---

## 2. 画面構成

```
┌─────────────────────────────────────┐
│ ← 利用規約                           │
├─────────────────────────────────────┤
│                                      │
│  利用規約                            │
│  （イントロテキスト）                │
│                                      │
│  第1条（適用範囲）                   │
│  本利用規約は、…                    │
│                                      │
│  第2条（利用者の義務）               │
│  ・正確な情報の提供                  │
│  ・アカウントの適切な管理            │
│  ・不正利用を発見した場合の通知      │
│                                      │
│  第3条（禁止事項）                   │
│  ・法令・公序良俗に反する行為        │
│  ・なりすまし行為                    │
│  ・サービスの妨害                    │
│  ・リバースエンジニアリング          │
│                                      │
│  第4条（免責事項）                   │
│  （本文）                            │
│                                      │
│  第5条（サービスの変更・終了）        │
│  （本文）                            │
│                                      │
│  第6条（準拠法・管轄）               │
│  （本文）                            │
│                                      │
│  ※ 本規約は予告なく変更する場合が   │
│  あります。                          │
├─────────────────────────────────────┤
│ 🏠    📋    ✅    🛒    ⚙️          │
└─────────────────────────────────────┘
```

---

## 3. 機能仕様

### 3.1 コンテンツ

以下の条項を順に表示する（静的テキスト・i18nキーより描画）:

| 条項 | 内容 |
|---|---|
| 第1条 | 適用範囲 |
| 第2条 | 利用者の義務（箇条書き） |
| 第3条 | 禁止事項（箇条書き） |
| 第4条 | 免責事項 |
| 第5条 | サービスの変更・終了 |
| 第6条 | 準拠法・管轄 |

末尾に改訂予告の注記を表示。

### 3.2 スクロール

長いコンテンツのためスクロール可能なレイアウトとする。

---

## 4. API

なし（この画面はAPIを呼び出さない）。
