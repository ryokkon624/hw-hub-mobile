# #13 買い物リスト モバイル仕様

## 1. 概要

世帯の買い物アイテムを管理する画面。未購入・かご・購入済みの3ステータスでアイテムを管理する。

モバイル版ではPC版の2カラムレイアウトをタブ切り替えに変更し、ステータス変更操作はスワイプで行う。

---

## 3. 機能仕様

### 3.1 タブ

| タブ | 表示内容 |
|---|---|
| 未購入 | 未購入アイテム一覧 + 購入場所フィルタ |
| かご | かごアイテム一覧 + 一括購入済みボタン |
| 購入済み | 直近7日間の購入済みアイテム（日付グループ） |

タブヘッダーに件数バッジを表示する。デフォルトは「未購入」タブ。

### 3.2 未購入タブ

**購入場所フィルタ**
- すべて / スーパー / ドラッグストア / オンライン
- 世帯に登録されている購入場所を動的に表示
- フロント側で絞り込み

**アイテムカード**

```
┌─────────────────────────────────┐
│ ▌ オリーブオイル          ☆  📷 │  ← ▌は購入場所カラー
│   メモ（任意）                   │
│  ← 削除           かごへ →     │  ← スワイプヒント
└─────────────────────────────────┘
```

| 要素 | 内容 |
|---|---|
| 購入場所カラーバー | カード左端に購入場所に対応した色のバーを表示 |
| 品名 | |
| メモ | 登録されている場合のみ表示 |
| お気に入りアイコン | ☆/★ タップでお気に入りトグル |
| 画像アイコン | 📷 画像が添付されている場合のみ表示。タップで#15詳細画面へ遷移 |

スワイプ操作:

| 操作 | 内容 |
|---|---|
| 右スワイプ | ステータスを「かご」に更新 |
| 左スワイプ | アイテムを削除（確認ダイアログを表示） |

カードタップで#15買い物アイテム詳細画面へ遷移。

### 3.3 かごタブ

**一括購入済みボタン（ヘッダー）**
- かご内の全アイテムを一括で購入済みに更新
- 会計完了後にまとめて更新するユースケースを想定
- タップで確認ダイアログを表示

**アイテムカード**

```
┌─────────────────────────────────┐
│ ▌ オリーブオイル          ☆  📷 │
│   メモ（任意）                   │
│  ← 未購入に戻す    購入済みに → │  ← スワイプヒント
└─────────────────────────────────┘
```

スワイプ操作:

| 操作 | 内容 |
|---|---|
| 右スワイプ | そのアイテムのみステータスを「購入済み」に更新 |
| 左スワイプ | ステータスを「未購入」に戻す |

> 💡 想定ユースケース: 商品を手に取るたびに「右スワイプでかごへ」→ 会計後に一括「購入済みにする」が基本操作。個別に購入済みにしたい場合は右スワイプで対応。

### 3.4 購入済みタブ

- 直近7日間の購入済みアイテムを日付グループで表示
- 各グループ: 日付 + 件数
- アイテム行: 品名 + 購入場所バッジ
- アイテム行タップで#15買い物アイテム詳細画面へ遷移
- スワイプ操作なし（購入済みからの変更不可）

## 4. API

既存APIをそのまま利用する。

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/households/{householdId}/shopping-items` | アイテム全件取得。フィルタはフロント側で実施 |
| PATCH | `/api/shopping-items/{shoppingItemId}/status` | ステータス更新（かごへ / 未購入に戻す / 購入済み） |
| PATCH | `/api/shopping-items/bulk-status` | かご内アイテムの一括購入済み更新 |
| PATCH | `/api/shopping-items/{id}/favorite` | お気に入りの切り替え |
| DELETE | `/api/shopping-items/{id}` | アイテム削除（未購入からの左スワイプ） |

## 5. 状態管理

```
ShoppingListState
├── items: List<ShoppingItem>       // 全アイテム（フィルタ前）
├── locationFilter: String          // 購入場所フィルタ（未購入タブ用）
├── activeTab: ShoppingTab          // unpurchased / cart / purchased
└── isLoading: bool
```

## 7. モバイル固有の考慮点

### スワイプ操作

**未購入タブ**:

| 操作 | 背景色 | ラベル |
|---|---|---|
| 右スワイプ（かごへ） | emerald-500（緑） | かごへ |
| 左スワイプ（削除） | rose-500（赤） | 削除 |

**かごタブ**:

| 操作 | 背景色 | ラベル |
|---|---|---|
| 右スワイプ（購入済みに） | emerald-500（緑） | 購入済みに |
| 左スワイプ（未購入に戻す） | slate-400（グレー） | 戻す |

**購入済みタブ**:
- スワイプ無効（購入済みからのステータス変更はしない）

- スワイプ中はカードが追従してドラッグし、しきい値（カード幅の30%）を超えたらアクション確定
- 購入場所ごとのカード背景色はスワイプ中も維持する
