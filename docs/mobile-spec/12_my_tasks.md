# #12 My Tasks モバイル仕様

## 1. 概要

自分に割り当てられている家事タスクの一覧画面。タスクの完了・スキップを登録する。

Web版SP版と同等の機能をそのまま踏襲する。

## 3. 機能仕様

### 3.1 過去の家事セクション

今日より前に対応する予定だった未対応タスクを日付ごとにグループ表示する。

| 要素 | 内容 |
|---|---|
| セクションヘッダー | ⚠️アイコン + 「過去の家事」ラベル + 説明文 |
| すべて完了にするボタン | タップで確認ダイアログを表示。確認後、過去の未対応タスクを全件完了に更新 |
| 日付グループ | 日付（曜日）+ 件数 |
| タスクカード | 家事名 + スキップボタン + 完了ボタン |

過去の未対応タスクが0件の場合、セクション自体を非表示にする。

### 3.2 これからの家事セクション

今日以降に予定されている自分担当のタスクを日付ごとにグループ表示する。

| 要素 | 内容 |
|---|---|
| セクションヘッダー | 「これからの家事」ラベル + 説明文 + 未対応件数 |
| フィルタ | すべて / 今日 / 1週間 の3種類（フロント側で絞り込み） |
| 日付グループ | 日付（曜日）+ 件数 |
| タスクカード | 家事名 + スキップボタン + 完了ボタン |

### 3.3 タスクカードの操作

| 操作 | 内容 |
|---|---|
| 完了ボタン | タスクのステータスを完了に更新。カードをリストから除去 |
| スキップボタン | タスクのステータスをスキップに更新。カードをリストから除去 |

## 4. API

既存APIをそのまま利用する。

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/housework-tasks` | タスク全件取得。自分のタスクのみ・フィルタはフロント側で実施 |
| PATCH | `/api/housework-tasks/{taskId}/status` | タスクステータス更新（完了 / スキップ） |
| PATCH | `/api/housework-tasks/bulk-status` | 過去タスクの一括完了 |

## 5. 状態管理

```
MyTasksState
├── pastTasks: List<HouseworkTask>    // 過去の未対応タスク
├── futureTasks: List<HouseworkTask>  // これからのタスク
├── filter: MyTasksFilter             // すべて / 今日 / 1週間
└── isLoading: bool
```
