# #19 世帯設定 モバイル仕様

## 1. 概要

所属世帯の一覧表示・切り替え・新規作成、選択中の世帯の名前変更・ニックネーム設定、メンバー管理（一覧・招待・削除・権限譲渡）、世帯削除を行う画面。

Web版SP版と同等の機能をそのまま踏襲する。

## 3. 機能仕様

### 3.1 世帯一覧・切り替え

- 所属世帯を一覧表示。現在の世帯にはバッジを表示
- タップで世帯を切り替え（store の `currentHouseholdId` を更新）
- 「新しいおうちを追加」タップで世帯作成ダイアログを表示

### 3.2 世帯名変更

- 必須・空文字不可
- 元の値から変更があった場合のみ「保存」ボタン活性

### 3.3 ニックネーム設定

- この世帯での表示名（家事割り当て画面などで使用）
- 最大50文字
- 入力済みの場合のみ「保存」ボタン活性

### 3.4 メンバー一覧

| ステータス | 表示 |
|---|---|
| 有効（ACTIVE） | 緑のバッジ |
| 招待中（INVITED） | 黄のバッジ |
| 離脱済み（LEFT） | グレーのバッジ |

**操作（OWNERのみ）:**
- 他の有効メンバーを「外す」→ 確認ダイアログ後に `DELETE /api/households/{id}/members/{userId}`
- 他の有効メンバーに「OWNERに譲渡」→ 確認ダイアログ後に `PUT /api/households/{id}/transfer-owner`

**自分自身（非OWNERのみ）:**
- 「この世帯から離脱」→ 確認ダイアログ後に `DELETE /api/households/{id}/members/me`

### 3.5 メンバー招待

- メールアドレス入力して「招待」タップ
- 送信済み招待の一覧（ステータス / 有効期限 / 参加者名を表示）
- PENDING 状態の招待: 「リンクをコピー」「取消」ボタン

**招待リンクのコピー:**
Universal Links方式を採用しているため、コピーするURLはhttpsのURLとする。アプリがインストールされていればアプリで開き、未インストールであればブラウザのWeb版で開く。

```
https://hwhub.familyapp-hwhub.com/invite/{token}
```

OSのシェアシートへ渡してコピー・共有できるようにする。

### 3.6 世帯削除（OWNERのみ）

- 自分以外の有効メンバーが存在する場合は削除不可
- 有効な家事マスタ件数・未購入の買い物件数を確認ダイアログに表示
- 確認後に `DELETE /api/households/{householdId}`

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/households/{householdId}/members` | メンバー一覧取得 |
| PUT | `/api/households/{householdId}` | 世帯名更新 |
| PUT | `/api/households/{householdId}/members/me/nickname` | ニックネーム更新 |
| DELETE | `/api/households/{householdId}/members/me` | 世帯から離脱 |
| DELETE | `/api/households/{householdId}/members/{userId}` | メンバー除外（OWNER操作） |
| PUT | `/api/households/{householdId}/transfer-owner` | OWNER譲渡 |
| GET | `/api/households/{householdId}/invitations` | 招待一覧取得 |
| POST | `/api/households/{householdId}/invitations` | 招待作成 |
| POST | `/api/household-invitations/{token}/revoke` | 招待取消 |
| POST | `/api/households` | 世帯新規作成 |
| DELETE | `/api/households/{householdId}` | 世帯削除 |
