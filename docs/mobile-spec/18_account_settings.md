# #18 アカウント設定 モバイル仕様

## 1. 概要

プロフィール（表示名・言語）、アイコン画像、パスワード変更、通知設定、Google連携、アカウント削除を管理する画面。

**モバイル固有の差異:**
- アイコン変更は `image_picker` でカメラ/フォトライブラリから選択
- 画像アップロードは S3 Presigned URL 方式（`POST /api/users/me/icon/upload-url` → S3 PUT → `POST /api/users/me/icon`）
- Google連携はネイティブ `google_sign_in` パッケージ経由（`POST /api/users/me/google/link/mobile`、モバイル向け新規エンドポイント）

## 3. 機能仕様

### 3.1 プロフィール設定（表示名・言語）

| フィールド | ルール |
|---|---|
| 表示名 | 必須、最大50文字 |
| 利用言語 | 日本語（`ja`）/ 英語（`en`）/ スペイン語（`es`）から選択 |

- 変更がある場合のみ「保存」ボタン活性
- 保存成功後スナックバーで通知

### 3.2 パスワード変更

| フィールド | ルール |
|---|---|
| 現在のパスワード | 必須 |
| 新しいパスワード | 必須、8文字以上 |
| 新しいパスワード（確認） | 新しいパスワードと一致 |

- Google連携のみのアカウント（パスワードなし）はこのセクションを非表示

### 3.3 プロフィール画像（モバイル固有）

1. 「カメラ」または「ライブラリ」ボタンをタップ
2. `image_picker` でカメラ撮影/フォトライブラリ選択
3. 選択した画像で即時アップロード（詳細は `common/image_upload.md` 参照）
4. アップロード完了後スナックバーで通知
5. 制限: 5MB以内、JPEG/PNG/GIF

### 3.4 通知設定

アプリ内のベルマーク通知（`/notifications`）のON/OFFを管理する設定。将来的にFCMプッシュ通知を追加する際もこの設定をそのまま流用できる設計とする。

| 項目 | 内容 |
|---|---|
| グローバルON/OFF | 全通知のON/OFF。OFFにすると個別設定も無効化 |
| 世帯関連 | 世帯メンバーの追加・削除通知 |
| タスク割り当て | タスクの担当者割り当て通知 |

- トグルは即時反映（保存ボタン不要）

### 3.5 Google連携（モバイル固有）

- メールアドレスが `@gmail.com` の場合のみセクションを表示
- **未連携時**: `google_sign_in` でネイティブ認証 → IDトークンを `POST /api/users/me/google/link/mobile` に送信
- **連携済み時**: 「連携済み」バッジと注意文を表示（解除操作は不可）

### 3.6 アカウント削除

- 確認ダイアログ表示後に実行
- 他のメンバーが存在する世帯のOWNERの場合は削除不可（エラー表示）

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| GET | `/api/users/me/profile` | プロフィール取得 |
| PUT | `/api/users/me/profile` | プロフィール更新（表示名・言語） |
| PUT | `/api/users/me/password` | パスワード変更 |
| POST | `/api/users/me/icon/upload-url` | アイコンアップロード用 Presigned URL 取得 |
| POST | `/api/users/me/icon` | アイコン更新（S3アップロード後のfileKey登録） |
| GET | `/api/users/me/notification-settings` | 通知設定取得 |
| PUT | `/api/users/me/notification-settings` | 通知設定更新 |
| POST | `/api/users/me/google/link/mobile` | Google連携（モバイル専用・新規エンドポイント） |
| DELETE | `/api/users/me` | アカウント削除 |
