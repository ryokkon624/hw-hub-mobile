# #28 プライバシーポリシー モバイル仕様

## 1. 概要

アプリのプライバシーポリシーを表示する静的コンテンツ画面。Web版SP版と同等の内容をそのまま踏襲する。APIコールなし。

## 3. 機能仕様

### 3.1 コンテンツ

以下のセクションを順に表示する（静的テキスト・i18nキーより描画）:

| セクション | 内容 |
|---|---|
| 1. 収集する情報 | アカウント情報 / 利用データ / ログ情報（箇条書き） |
| 2. 情報の利用目的 | サービス提供 / 不具合調査 / 分析（箇条書き） |
| 3. 第三者提供 | 原則として第三者提供しない旨を説明 |
| 4. 安全管理措置 | 適切な安全管理措置を講じる旨を説明 |
| 5. 開示・訂正・削除 | ユーザーからの請求手続きを説明 |
| 6. お問い合わせ | 問い合わせ窓口の案内 |

末尾に改訂予告の注記を表示。

### 3.2 スクロール

長いコンテンツのためスクロール可能なレイアウトとする。
