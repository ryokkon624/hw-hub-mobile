# #15 買い物アイテム詳細 モバイル仕様

## 1. 概要

買い物アイテムの詳細確認・編集画面。画像の追加・削除、お気に入り登録、ステータス変更、アイテム削除が可能。

Web版と同等の機能をそのまま踏襲する。

## 3. 機能仕様

### 3.1 フォーム

| フィールド | 内容 |
|---|---|
| 品名 | テキスト入力（必須） |
| メモ | テキスト入力（複数行・任意） |
| 購入場所 | プルダウン選択 |
| お気に入り | ★アイコンタップでトグル |

### 3.2 ステータス変更

ステップ形式（1 未購入 → 2 かご → 3 購入済み）で現在のステータスを表示。タップで変更可能。

| ステータス | 説明 |
|---|---|
| 未購入 | まだ買う前の状態 |
| かご | 買い物かごに入れた状態 |
| 購入済み | 購入完了 |

### 3.3 画像

- 登録済み画像を横スクロールで表示
- 各画像に削除ボタンを表示
- 「画像を追加」エリアから追加可能（image_pickerでカメラ/フォトライブラリから選択）
- 1枚ずつ追加（上限枚数はWeb版の実装に準ずる）

**画像追加の処理フロー**
1. image_pickerで画像選択
2. Presigned URL取得 `POST /api/shopping-items/{itemId}/attachments/upload-url`
3. S3へ直接PUT
4. メタデータ登録 `POST /api/shopping-items/{itemId}/attachments`
5. 画像一覧を再取得

### 3.4 保存

「保存する」ボタンタップで以下を実行。

1. バリデーション（品名必須）
2. アイテム更新 `PATCH /api/shopping-items/{itemId}`
3. 完了後、#13買い物リスト画面に戻る

### 3.5 削除

「このアイテムを削除」ボタンタップで確認ダイアログを表示。確認後、アイテムを削除して#13買い物リスト画面に戻る。

## 4. API

| メソッド | エンドポイント | 用途 |
|---|---|---|
| PUT| `/api/shopping-items/{shoppingItemId}` | アイテム更新 |
| DELETE | `/api/shopping-items/{shoppingItemId}` | アイテム削除 |
| GET | `/api/shopping-items/{shoppingItemId}/attachments` | 画像一覧取得（Presigned GET URL含む） |
| POST | `/api/shopping-items/{itemId}/attachments/upload-url` | 画像アップロード用Presigned URL発行 |
| POST | `/api/shopping-items/{itemId}/attachments` | 画像メタデータ登録 |
| DELETE | `/api/shopping-items/{itemId}/attachments/{attachmentId}` | 画像削除 |
