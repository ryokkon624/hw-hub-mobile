# #17 設定トップ モバイル仕様

## 1. 概要

設定画面への入口となるナビゲーションメニュー画面。Web版SP版と同等の機能をそのまま踏襲する。

---

---

## 3. 機能仕様

各カードをタップすると対応する設定画面へ遷移する。

| カード | 遷移先 |
|---|---|
| アカウント設定 | #18 アカウント設定 |
| 世帯設定 | #19 世帯設定 |
| 家事設定 | #20 家事設定一覧 |
| 通知センター | #16 通知センター |
| お問い合わせ | #23 問い合わせ一覧 |
| アプリ情報 | #26 アプリ情報 |

---

## 4. API

なし（この画面はAPIを呼び出さない）。
